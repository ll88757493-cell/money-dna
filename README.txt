GitHub Pages 上傳方式：
1. 解壓縮 money-mirror-3-5-beta.zip
2. 到 GitHub Repository
3. 上傳/覆蓋以下資料：
   - index.html
   - css 資料夾
   - js 資料夾
4. Commit changes
5. 等待 GitHub Pages 自動更新

LINE 按鈕設定：
打開 js/app.js
把這行：
const LINE_URL = "https://line.me/R/ti/p/@你的LINE官方帳號";
